Quick start

Short User Manual

  1. Start the backend locally or download a packaged release for your operating system.
  2. Open the browser UI at /web or launch the macOS app archive on Apple Silicon Macs.
  3. Fill in the search fields, choose a summary length, and run Create Summary.
  4. Review the returned messages and summary, then use Mark Read, Tag Summarised, or Email Summary if needed.
  5. Open the Log tab to review activity and use row-level Undo where it is available.
  6. Open Settings to switch model providers, configure provider keys, and point the UI at a different backend URL.
