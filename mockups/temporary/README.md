# Temporary Mockups (Approval Stage)

These are temporary, non-functional concept mockups for UX/UI approval.

## Files

- `index.html`: all proposed screens in one browsable page
- `styles.css`: visual system for mockups only

## Preview

From repo root:

```bash
python -m http.server 8012 --directory mockups/temporary
```

Then open:

- `http://127.0.0.1:8012`

## Scope represented

- Web Main (Inbox Digest Studio)
- Web Settings (split IA)
- Web Log (timeline)
- macOS Main concept
- macOS Settings concept
- macOS Log concept

## Approval checklist

- [ ] Information architecture direction approved
- [ ] Visual language direction approved
- [ ] Interaction model direction approved
- [ ] Prioritized implementation order approved

No production UI behavior is changed by these files.
